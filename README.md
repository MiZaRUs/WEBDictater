## Простой диктофон с минималистичным редактором текста
### Это неполноценный сервис, а костыль к серверу whisper.cpp, создан на основе многих похожих проектах.

Приложение в виде WEB-Сервиса, позволяет надиктовать текст и тут-же поправить его.
Для перевода аудио в текст используется сервис whisper.cpp

build_srv_in_docker.sh - сборка в docker-контейнер.
